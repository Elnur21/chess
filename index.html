Chess Game

This is a simple chess application where you can play against the computer or challenge another player.

How it works

Choose your preferred mode and start playing instantly. The game is designed to be easy to understand while still offering a fun experience.

Single Player

  • Play against the built-in AI
  • Select an AI difficulty level
  • Use the timer to add extra challenge

Multiplayer

  • Connect to another player
  • Play live over the network
  • Watch moves update in real time

Game Features

  • Leaderboard with player points
  • Move history panel
  • Timer and game controls

Tip: Start by entering a username, then choose whether you want to play alone or with another player.
